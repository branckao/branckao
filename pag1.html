Serviços para o transportador

Oferecemos diversas opções para assegurar as cargas dos transportadores. Em cada atendimento, nosso foco é entender as necessidades da empresa e quais as coberturas mais adequadas para cada uma delas. Trabalhamos com os seguintes seguros para o transportador:

  • Seguro obrigatório de responsabilidade civil (RCTR-C)
  • Seguro de roubo de carga (RCF-DC)
  • Seguro para cargas aéreas (RCTA-C)
  • Seguro de responsabilidade para o armador de carga
  • Seguro de cargas internacionais - Mercosul
  • Seguro de Garantia Aduaneira
